HTML

  • The head element includes info about the webpage
  • The body element includes the content visible to the user
  • HTML uses the id attribute to identify elements

CSS

  • CSS stands for Cascading Style Sheets
  • Three types of CSS: inline, embedded and external CSS
  • The box model consists of margins, borders, padding and content

Git

  • Git is free and open source software for version control
  • GitHub is a code hosting platform for version control and collaboration
  • Branch: workspace where you can make changes that don't affect the live site

JavaScript

  • The console.log() method outputs a message the the web console
  • A string is a piece of text
  • Arrays are used to store multiple values in a single variable
